## 📋 What is java-command-handler?

The java-command-handler application shows a way to organize commands inside a Java program. It helps separate different command actions so the code stays clear and easy to manage. This set of examples and tools follows a design style known as the command handler pattern. It is useful for building programs that need clear steps for commands, making them easier to change or grow later.

You do not need to know programming to use this. This guide will help you get the application running on Windows. Once set up, you can see how commands work inside a Java application in a practical way.

## ⚙️ Running the Application on Windows

To run the java-command-handler application, you need to use the Command Prompt. This might sound technical, but do not worry. Follow these directions step by step.

1. Open the Start menu and type **cmd** to find the Command Prompt. Click to open it.
2. In the Command Prompt window, go to the folder where you extracted the files. You do that by typing:

   ```
   cd Desktop\java-command-handler-main
   ```

   Replace `Desktop` and the folder name if you extracted the ZIP somewhere else or the folder has a different name.

3. Once inside the folder, type this command to compile the Java code:

   ```
   javac *.java
   ```

4. If there are no errors, run the application by typing this:

   ```
   java Main
   ```

The screen should show simple instructions or examples of commands running in the app.

## 🔧 Troubleshooting Common Issues

If you encounter problems, try these tips:

- Make sure Java is installed and added to your system’s PATH. You can check by typing `java -version` in Command Prompt.
- Double-check you are inside the right folder using the `cd` command.
- When compiling, ensure all Java files are in the same folder.
- If you see "command not recognized" errors, Java might not be installed or correctly set up.
- Restart the Command Prompt if commands don’t seem to work.
- If the application runs but shows no output, try typing `java Main` again or check that all files were extracted correctly.
